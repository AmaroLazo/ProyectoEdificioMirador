Version 1.0

Funciones implementadas:

  -Se implemento un Login básico (Correo y Contraseña) junto con su botón de iniciar sesión.
  
  -Se implemento un dashboard general que muestra las cuentas pendientes del usuario.
  
  -Se implemento flujo de pago con multiples opciones de pago.
  
  -Se implemento un botón de pagar todas las cuentas.
  
  -Se implemento un botón de cerrar sesión.
  
  -Se implemento la creación de boleta al concluir el pago

  -Botón de "Pagar Todo" no funcionaba correctamente.
  
  -No Muestra opción de cambio de propiedad en caso de que el propietario tenga mas de una.
  
  -Botón de acceso Admin visible a todos los usuarios.
